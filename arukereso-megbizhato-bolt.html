A bővítmény legfőbb jellemzői

🔑 Árukereső WebAPI integráció

A vásárlások utáni kérdőív adatokat automatikusan továbbítja az Árukereső felé. Csak aktiválnod kell a funkciót az Árukereső adminban, és beilleszteni a WebAPI kulcsot a plugin beállításaiba.

✉️ Testreszabható küldő e-mail & SPF

A véleménybekérő kérdőívek kiküldhetők a saját domain e-mail címedről (az _spf-c.arukereso.hu SPF rekord beállításával), vagy automatikusan a megbizhatobolt@arukereso.hu címről.

⚡ Ingyenes letöltés & CodeStudio API

A bővítmény ingyenesen letölthető és telepíthető bármelyik WooCommerce webáruházba. A működéséhez és élesítéséhez egy CodeStudio API kulcs megvásárlása szükséges.

CodeStudio API kulcs vásárlása

Töltsd le a plugint ingyenesen, majd kérj CodeStudio API kulcsot a bővítmény aktiválásához!
